## Workflow

A quiz software

1. Institute admin can register an institute
2. Institute admin can create participants
3. Institute admin can create quizzes
4. Institute admin can assign quizzes to participants
5. Participant can take quizzes
6. Participant can view results
7. Institute admin can view results
8. Institute admin can view institute results
9. System admin can view all results
10. Institute admin can view institute participants
11. Institute admin can view institute quizzes
12. Institute admin can view institute participants results
13. Institute admin can view institute participants attempts

use livewire 4 with blaze by default. 

Use 3 different layouts for admin, institute and participant.

Admin layout should have a sidebar on left and top bar on top, and main content on right.
Institute layout should have a sidebar on left and top bar on top, and main content on right.
Participant layout should have a sidebar on left and top bar on top, and main content on right.

also some layouts for quiz, auth, too
